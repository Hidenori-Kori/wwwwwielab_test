C/C++環境の導入

C/C++言語を利用するにはコンパイラの利用が必須になります。
本研究室では、デバッガやエディタが統合された開発環境（IDE）である Visual Studio Community の利用を推奨しています。

1. インストーラーのダウンロード

Microsoftの公式サイトから、無料版である Visual Studio Community をダウンロードします。

  • Visual Studio のダウンロード (Microsoft 公式)

「コミュニティ」を選択してインストーラー（.exeファイル）をダウンロードし、起動してください。

2. 重要：ワークロードの選択

インストーラーが起動すると、インストールする機能を選択する画面になります。
ここで必ず「C++ によるデスクトップ開発」にチェックを入れてください。

注意： これにチェックを入れないと、Visual Studioはただのエディタとなり、C言語のコンパイルができません。

チェックを入れたら、右下の「インストール」ボタンをクリックします。インストールには時間がかかります（数GBあります）。

3. 動作確認（Hello World）

インストールが完了したらVisual Studioを起動し、「新しいプロジェクトの作成」を選択します。

プロジェクト設定

  1. 「空のプロジェクト (C++)」を選択して「次へ」。
  2. プロジェクト名（例: TestProject）を入力して「作成」。

ソースコードの追加

右側の「ソリューションエクスプローラー」で、「ソースファイル」を右クリック → 「追加」 → 「新しい項目」を選択し、main.cpp を作成します。

以下のコードを貼り付けてください。

#include <stdio.h>
                    int main() {
                        printf("Hello, World!\n");
                        return 0;
                    }

上部の緑色の再生ボタン（ローカル Windows デバッガー）を押して、黒い画面に "Hello, World!" と表示されれば環境構築は成功です。
