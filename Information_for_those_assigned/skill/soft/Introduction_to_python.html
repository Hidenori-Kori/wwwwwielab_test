Python入門

講義資料「Pythonを用いたプログラミング言語入門」に基づく基礎文法と、研究で頻出するライブラリのまとめです。

1. 基本構文とデータ型

基本ルール

  • インデント (字下げ): if文やループ文のブロックは、インデントを揃えることで定義されます (オフサイドルール) 。
  • コメント: # から行末まではコメントになります [cite: 54]。
  • 変数宣言: 型宣言は不要で、代入することで自動的に型が決まります [cite: 88]。

主なデータ型

型名 Python表記 説明
整数型 int 扱える値に制限はありません [cite: 131]。
浮動小数点型 float 実数を扱います [cite: 131]。
文字列型 str ' または " で囲みます [cite: 131, 139]。
ブール型 bool True または False (条件判定用) [cite: 131]。
リスト list [ ] で定義。要素の追加・変更が可能 。
タプル tuple ( ) で定義。作成後の変更不可 。

算術演算子 [cite: 164]

記号意味例

//商 (整数)8 // 5 → 1

3. 制御構造

分岐 (if) [cite: 265, 313]

            if 条件式1:
                処理1
            elif 条件式2:
                処理2
            else:
                処理3

繰り返し (for / while)

回数指定 (for): range(開始, 終了, ステップ) を使用します [cite: 408, 430]。

            for i in range(10):  # 0から9まで繰り返す
                print(i)

条件指定 (while): 条件が True の間繰り返します [cite: 446]。

            while x <= 100:
                sum += x
                x += 1

※ ループ制御には break (中断) [cite: 616] や continue (スキップ) [cite: 631] が使えます。

4. 関数とファイル操作

関数の定義 [cite: 244]

            def 関数名(引数1, 引数2):
                処理
                return 戻り値

ファイルの読み書き [cite: 648, 653]

            # 読み込み ('r')
            f = open("data.txt", "r", encoding="utf-8")
            content = f.read()
            f.close()

            # 書き込み ('w':上書き, 'a':追記)
            f = open("result.txt", "a")

            f.close()

5. 科学技術計算ライブラリ

研究では以下のライブラリを組み合わせて使用します。

NumPy (数値計算) [cite: 560]

高速な行列演算やベクトル計算を行うための基本ライブラリです。講義資料では行列の転置やゼロ行列の生成に使用されています。

            import numpy as np  # 慣例として np と略す

            # 行列の定義 (多次元リストの代わり)
            A = np.array([[1, 2], [3, 4]])

            # ゼロ行列 (初期化などに便利) [cite: 579]
            Z = np.zeros(5) 

            # 転置行列 [cite: 564]
            A_t = np.transpose(A)

            # 平方根 [cite: 813]
            root = np.sqrt(2)

SciPy (高度な科学技術計算)

NumPyをベースに、信号処理、最適化、統計、積分などの高度な科学計算機能を提供します。

  • 用途: 微分方程式の解法、最適化問題 (最小二乗法など)、フーリエ変換 (FFT)。
  • 例: from scipy import optimize (最適化)

Matplotlib (グラフ描画)

実験データの可視化やシミュレーション結果のグラフ作成に使用します。

            import matplotlib.pyplot as plt

            x = np.linspace(0, 10, 100)
            y = np.sin(x)

            plt.plot(x, y)
            plt.xlabel("Time [s]")
            plt.ylabel("Voltage [V]")
            plt.show()

Pandas (データ解析)

CSVファイルの読み込みや、表形式のデータの操作に特化しています。

            import pandas as pd

            # CSVの読み込み
            df = pd.read_csv('experiment_data.csv')

            # データの表示
            print(df.head())
