C/C++入門

研究活動（数値計算・制御・組み込み）で頻出するC/C++言語の基礎構文とライブラリのまとめです。

1. 基本構造 (C vs C++)

C言語とC++は互換性がありますが、入出力の書き方が異なります。

C言語 (stdio.h)

                            #include <stdio.h>

                            int main() {
                                // コンソール出力
                                printf("Hello, World!\n");
                                return 0;
                            }

C++ (iostream)

                            #include <iostream>

                            int main() {
                                // コンソール出力
                                std::cout << "Hello, World!" << std::endl;
                                return 0;
                            }

2. データ型と書式指定子

printf で変数を表示する際、型に対応した書式指定子が必要です。

型名 サイズ (例) 書式指定子 (printf) 用途
int 4 byte %d 整数
float 4 byte %f 単精度実数
double 8 byte %lf 倍精度実数 (研究ではこちらを推奨)
char 1 byte %c 1文字
char[] - %s 文字列

3. 配列とポインタ (重要)

C/C++最大の特徴であるメモリアドレスの扱いについてです。

配列

                    int a[5] = {10, 20, 30, 40, 50};
                    // a[0] が 10, a[4] が 50

ポインタ

変数の「アドレス（メモリ上の場所）」を格納する変数です。

  • &変数 : 変数のアドレスを取得 (アドレス演算子)
  • *ポインタ : アドレスの中身にアクセス (間接参照演算子)
                    int x = 100;
                    int *ptr = &x;  // xのアドレスをptrに代入

                    printf("%d\n", *ptr); // アドレスptrの中身(100)を表示

4. 制御構造

条件分岐 (if / switch)

                    if (x > 0) {
                        printf("Positive\n");
                    } else if (x == 0) {
                        printf("Zero\n");
                    } else {
                        printf("Negative\n");
                    }

繰り返し (for / while)

                    // カウンタによるループ
                    for (int i = 0; i < 10; i++) {
                        printf("i = %d\n", i);
                    }

                    // 条件によるループ
                    while (x < 100) {
                        x *= 2;
                    }

5. 関数の定義

処理をまとめるブロックです。使用する前に宣言（プロトタイプ宣言）するか、main関数の上に書く必要があります。

                    // 戻り値の型 関数名(引数)
                    double square(double x) {
                        return x * x;
                    }

                    int main() {
                        double result = square(2.5);
                        return 0;
                    }

6. 研究で役立つライブラリ

数学関数 (math.h / cmath)

コンパイル時に -lm オプションが必要な場合があります (Linux/gcc)。

関数説明例
sqrt(x)平方根 \(\sqrt{x}\)sqrt(2.0)
pow(x, y)べき乗 \(x^y\)pow(2.0, 3.0)
sin(x), cos(x)三角関数 (ラジアン)sin(3.14159)
exp(x)指数関数 \(e^x\)exp(1.0)
log(x)自然対数 \(\ln x\)log(10.0)
fabs(x)絶対値 \(|x|\)fabs(-5.5)

C++の標準ライブラリ (STL)

C++を使う場合、動的配列 vector などが非常に便利です。

                    #include <vector>
                    #include <algorithm> // ソートなどに使用

                    std::vector<int> v = {1, 5, 3};
                    v.push_back(10); // 末尾に追加
                    std::sort(v.begin(), v.end()); // 並び替え
