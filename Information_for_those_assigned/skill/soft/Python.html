Python環境の導入

魏研究室では最適化設計に用いるプログラミング言語としてC/C++を主に使用しますが、
データの事後処理、グラフ描画、あるいは特定アルゴリズムのプロトタイピングに Python を使用します。
ここでは環境変数の設定（Path通し）が不要で、管理が容易な方法で環境を構築します。

1. Pythonのインストール (Microsoft Store)

Windows環境では、Microsoft Storeからインストールするのが最も手軽でトラブルが少ない方法です。
環境変数（Path）を自動で管理してくれます。

  1. Windowsのスタートメニューから「Microsoft Store」を開きます。
  2. 検索バーに「Python 3.13」（またはその時点の最新安定版）と入力します。
  3. 「入手」または「インストール」をクリックします。

2. VS Codeの拡張機能導入

エディタとして Visual Studio Code (VS Code) を使用します。
VS Codeを起動し、Python開発に必要な拡張機能をインストールします。

  1. VS Code左側のアイコン群から「拡張機能 (Extensions)」をクリックします（ショートカット: Ctrl + Shift + X）。
  2. 検索窓に「Python」と入力します。
  3. Microsoft製の「Python」拡張機能を選択し、「インストール」をクリックします。

3. インタープリンタの設定と実行確認

Pythonが正しく認識されているか確認し、簡単なプログラムを実行します。

ファイルの作成

VS Codeで新しいファイルを作成し、hello.py という名前で保存します。
以下のコードを入力してください。

print("Hello, Wei Lab!")

重要：インタープリンタの選択

VS Codeが「どのPythonを使って実行するか」を認識させる必要があります。

  1. 画面右下のステータスバーにある「{ } Python ...」のような表示、または「インタープリタを選択」をクリックします。
    （表示がない場合は Ctrl + Shift + P を押し、「Python: Select Interpreter」と入力して選択します）
  2. リストから、先ほどMicrosoft Storeでインストールした Python 3.xx (Microsoft Store) を選択します。

実行

右上の「▷（実行ボタン）」を押すか、F5 キーを押します。
下部のターミナルに Hello, Wei Lab! と表示されれば成功です。

4. 研究用ライブラリのインストール

研究では数値計算やグラフ描画を行うため、以下の外部ライブラリをインストールしておきましょう。

  • NumPy / SciPy: 数値計算、最適化計算用
  • Matplotlib: グラフ描画用

VS Codeの下部にある「ターミナル」タブをクリックし、以下のコマンドを入力してEnterキーを押してください。

pip install numpy scipy matplotlib pandas

エラーが出ずにインストールが完了（Successfully installed...と表示）すれば、環境構築は全て完了です。
